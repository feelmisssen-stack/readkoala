# 도란서재

도란서재는 초등학생들이 책을 읽고 **감상을 글로 남기며**, 친구들의 독서 기록에서 영감을 받고, **글을 쓸수록 코알라가 자라나는** 독서·감상 공유 웹앱입니다.

학생은 교사(관리자)가 안내한 아이디와 비밀번호로 로그인하여 책을 등록하고, 단계별 감상 기록·낱말집·도란뜰(채팅) 등을 이용합니다. 감상 글과 공유 문장은 Firebase Firestore에 저장되며, 기억에 남는 장면 그림은 Firebase Storage에 저장됩니다. 교사는 관리자 화면에서 회원·콘텐츠·채팅 승인·학습 기록을 확인하여 수업과 지도에 활용할 수 있습니다.

## 2. 개발 목적

도란서재는 학생들이 독서 후 느낀 생각을 **한 번에 긴 글**로 쓰기 어렵다는 점을 고려하여, **읽기 전·읽는 중·읽은 후** 질문과 짧은 기록으로 나누어 부담 없이 글쓰기를 시작하도록 돕기 위해 개발한 웹앱입니다.

초등학생들은 책을 읽는 것에서 그치기 쉽지만, 질문하기·연상하기·인상 깊은 문장 남기기·감상문 쓰기·장면 그리기 같은 **단계적 활동**을 통해 독서 경험을 깊게 만들 필요가 있습니다. 또한 친구들의 감상을 보며 읽기 동기를 키우고, 낱말과 문장을 자신의 것으로 확장하는 경험도 중요합니다.

이 웹앱은 학생들이 **자신의 책장**을 꾸리고, **감상 기록**을 쌓으며, **잎새와 코알라 성장**으로 성취감을 느끼고, **친구 피드·도란뜰·낱말집**을 통해 독서 공동체 안에서 성장할 수 있도록 구성하였습니다.

## 3. 주요 기능

### 3-1. 학생 로그인

학생은 교사가 발급하거나 안내한 **아이디와 비밀번호**로 로그인합니다.

웹에서 **자가 회원가입은 지원하지 않으며**, 계정은 관리자(교사)가 생성합니다. 로그인 후 상단 바에 **Lv. N · 닉네임 · 로그아웃(아이콘)** 이 표시됩니다. Lv는 감상·공유 문장 글자 수에 따른 코알라 성장 단계입니다.

**뷰어계정:** 사이트를 미리 둘러볼 때는 아래 계정을 사용할 수 있습니다. 홈·내 책장·감상 작성 화면 등 **모든 화면을 탐색**할 수 있지만, 책·감상·낱말·채팅 등 **저장·등록·수정·삭제는 할 수 없습니다.** 로그인 시 상단에 뷰어계정 안내 배너가 표시됩니다.

| 아이디 | 비밀번호 | 용도 |
|--------|----------|------|
| `tester` | `test1234` | 뷰어계정 |

### 3-2. 내 책장과 독서 온도계

학생은 읽은 책을 등록하고 **독서 진행률(독서 온도계)** 을 관리합니다.

ISBN 입력, 바코드 스캔, 도서관 정보나루·Google Books·Open Library(·알라딘) 검색으로 책을 추가할 수 있습니다. 책장 상단에는 **코알라 성장 카드**(Lv, 10칸 그래프, 잎새 수, 단계 이름·설명)가 표시됩니다.

### 3-3. 감상 기록하기

책마다 **단계별 감상 기록**을 작성합니다.

| 섹션 | 내용 |
|------|------|
| 읽기 전 질문 | 책을 읽기 전 궁금한 점 |
| 읽는 중 질문 | 읽으면서 생긴 질문 |
| 연상 문장 | 책을 읽으며 떠오른 생각 |
| 책속 한마디 | 인상 깊은 문장 인용 |
| 감상문 | 제목·이유·내용·인상 장면·느낀 점 |
| 기억에 남는 장면 | 장면 그림 업로드 |

작성 여부에 따라 카드 색이 달라지며, **감상문** 작성 시 **AI 감상문 도우미**(Gemini API, 키 없으면 정적 힌트)를 이용할 수 있습니다. 장면 그림은 업로드 후 AI·관리자 검토를 거칠 수 있습니다.

### 3-4. 코알라 10단계 성장

학생이 쓴 감상·공유 문장의 글자 수에 따라 **잎새**가 쌓이고 **Lv 1~10** 코알라로 성장합니다.

- 10바이트 = 잎새 1개  
- 잎새 100개 = Lv 1단계 상승 (최대 Lv 10)  
- 단계별 일러스트: `public/images/koala-stages/koala-lv1.png` ~ `koala-lv10.png`

### 3-5. 홈 — 친구 감상 모자이크 피드

로그인 후 홈에서 친구들이 남긴 감상 조각을 **책 표지 모자이크** 형태로 볼 수 있습니다.

읽기 전/중 질문, 연상, 책속 한마디, 기억에 남는 장면 등이 주기적으로 바뀌며 표시됩니다(모바일 3열·데스크톱 4열 그리드). 비로그인 시 로그인 폼이 표시됩니다.

### 3-6. 낱말집·국어사전·초성 퀴즈

**낱말집** 메뉴에서 표준국어대사전 API로 낱말을 검색·저장하고, **초성 퀴즈**로 복습하며, **낱말 하나, 문장 하나**로 문장을 만들어 공유할 수 있습니다.

사전 API의 `인공^지능` 같은 형태소 표기(`^`)는 앱에서 **`인공지능`** 으로 표시·저장합니다.

### 3-7. 도란뜰(책 이야기방)

책과 관련된 **이야기방**에 참여하여 메시지를 주고받을 수 있습니다.

방 개설·참여는 **관리자 승인**이 필요합니다. 메시지에 하트(공감)를 남길 수 있습니다.

### 3-8. 교사(관리자) 기능

교사는 **Google 계정**으로 `/admin`에 로그인하여 다음을 관리할 수 있습니다.

- 학생 계정 생성·수정·삭제  
- 도란뜰(채팅방) 승인  
- AI 도우미 세션·안전 검토(욕설·부적절 표현)  
- 공감(empathy) 등 운영 데이터 확인

## 4. Firebase 백엔드 활용

본 웹앱은 Firebase를 기반으로 회원·독서·감상 데이터를 처리합니다.

`NEXT_PUBLIC_FIREBASE_*`와 `FIREBASE_ADMIN_*` 환경 변수가 모두 설정되어야 로그인과 데이터 저장이 정상 동작합니다.

| 저장소 | 주요 데이터 |
|--------|-------------|
| **Firestore** | 회원, 책, 감상문, 낱말집, 초성 퀴즈 진행, 공유 문장, 채팅, 공감, 검토·AI 로그 |
| **Firebase Storage** | 기억에 남는 장면 그림 파일 |

저장된 정보는 교사가 학생의 독서·글쓰기 활동을 파악하고 수업·지도에 활용할 수 있습니다.

`data/db.json`은 예전 낱말집 1회 이전·동기화 스크립트용으로만 남아 있습니다.

## 5. 화면 구성

### 첫 화면(홈)

- 로그인하지 않은 경우: 로그인 폼  
- 로그인한 경우: 친구 감상 **모자이크 피드**  
- 하단(로그인 후): 이용약관·개인정보처리방침 링크  

### 상단 내비게이션

| 메뉴 | 설명 |
|------|------|
| 홈 | 친구 감상 피드 |
| 내 책장 | 코알라 성장 카드, 등록 책 목록 |
| 도란뜰 | 책 이야기방 |
| 낱말집 | 사전·낱말집·초성 퀴즈 |

모바일에서는 탭 메뉴가 가로 스크롤됩니다.

### 학생 학습·기록 화면

- **내 책장** → 책 선택 → **감상 기록하기** → 섹션별 작성 화면  
- **낱말집** → 검색·저장·퀴즈·문장 공유  
- **설정** → 닉네임 등 회원 정보 수정  

### 교사 관리 화면

- `/admin` — Google 로그인 후 사용자·채팅·검토·AI 로그 관리

## 6. 사용 방법

1. 웹 브라우저에서 아래 주소에 접속합니다.  
   **https://doranbooks.vercel.app**

2. 학생은 교사가 안내한 **아이디와 비밀번호**로 로그인합니다. 사이트를 먼저 둘러보기만 하려면 **뷰어계정** `tester` / `test1234` 로 로그인할 수 있습니다(저장·등록 불가).

3. **내 책장**에서 책을 검색·등록합니다.

4. 책을 선택하여 **감상 기록하기**에서 읽기 전/중 질문, 연상, 한마디, 감상문, 장면 등을 작성합니다.

5. 글을 쓸수록 **잎새**와 **코알라 Lv**가 올라갑니다. 내 책장 상단에서 확인합니다.

6. **홈**에서 친구들의 감상을 보고, **낱말집**에서 낱말을 모으며, **도란뜰**에서 이야기를 나눕니다.

7. 감상·낱말·채팅 기록은 Firebase에 저장되며, 교사는 관리자 화면에서 확인할 수 있습니다.

## 7. 개인정보 보호 및 계정 안내

본 웹앱은 **실제 수업 및 학습 활동**에 사용할 수 있도록 구성된 교육용 웹앱입니다.

- 학생 계정과 교사 Google 계정은 **개인정보 보호와 계정 보안**을 위해 공개하지 않습니다.  
- **뷰어계정**(`tester` / `test1234`)만 예외적으로 공개하며, 실제 학생 정보가 연결되지 않고 데이터를 저장할 수 없습니다.  
- 실제 학생 계정에는 학생 식별 정보와 학습·감상 기록이 연결될 수 있으므로, **공개 게시판 또는 GitHub README에는 실제 학생 계정 정보를 제공하지 않습니다.**  
- **만 14세 미만** 아동은 학교 **가정통신문(개인정보 수집·이용 동의서)** 등을 통해 법정대리인의 동의를 받은 후 이용합니다.  
- 학생 계정은 **교사(관리자)가 생성·배부**하며, 탈퇴·삭제는 담당 교사에게 요청합니다.

## 10. 기대 효과

도란서재는 학생들이 독서 후 **질문하고, 연상하고, 글로 표현하는** 경험을 제공한다는 점에서 **독서·글쓰기·의사소통** 역량 향상에 도움을 줄 수 있습니다.

또한 **잎새·코알라 성장**으로 기록의 양과 성취를 시각화하고, **친구 피드·도란뜰**을 통해 독서 공동체 안에서 동기를 키울 수 있습니다. 교사는 관리자 화면과 Firestore에 저장된 기록을 바탕으로 학생별 독서·감상 활동을 확인하여 **수업 설계와 개별 지도**에 활용할 수 있습니다.
